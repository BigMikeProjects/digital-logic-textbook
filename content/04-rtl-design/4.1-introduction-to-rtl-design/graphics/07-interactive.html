State Diagram + Datapath
💡
An FSMD (Finite State Machine with Datapath) couples a controller FSM with datapath operations. Each state issues control signals that direct the datapath (load, increment, select), while input signals determine which state transition the FSM takes.

FSM Controller → Datapath
💡
The FSM controller reads external inputs (ld, sel) and its current state to produce control signals (mux_sel, Y_en) that drive the datapath hardware. The datapath contains the MUX, adder, and register that perform the actual computation.
